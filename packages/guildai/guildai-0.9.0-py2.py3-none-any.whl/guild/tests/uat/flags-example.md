# Flags example

    >>> cd(example("flags"))

## Args

    >>> run("guild run args-default -y")
    Flags: 1 1.1 False hello
    <exit 0>

    >>> run("guild run args-default i=9 f=9.9 s=hola -y")

    <exit 0>

## Config

YAML based:

    >>> run("guild run config -y")
    Resolving config:flags.yml
    {'b': False,
     'f': 1.123,
     'i': 123,
     'l': [1, 1.2, 'blue', True],
     's': 'Howdy Guild'}
    <exit 0>

    >>> run("guild run config b=yes i=321 l='2 2.3 green no' -y")
    Resolving config:flags.yml
    {'b': True,
     'f': 1.123,
     'i': 321,
     'l': [2, 2.3, 'green', False],
     's': 'Howdy Guild'}
    <exit 0>

    >>> run("guild cat -p flags.yml")
    b: true
    f: 1.123
    i: 321
    l:
    - 2
    - 2.3
    - green
    - false
    s: Howdy Guild
    <exit 0>
